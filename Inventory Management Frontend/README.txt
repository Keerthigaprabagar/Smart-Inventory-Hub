Inventory Frontend (Angular 20)
This is the frontend for my project "Smart Inventory Management System" using ASP.NET core Web API(backend) and Angular20(frontend)

 In the project root run:
   npm install
 Start the dev server:
   npx ng serve --open
 The app expects the backend API at http://localhost:5160 (update src/app/services/api.service.ts if different).
